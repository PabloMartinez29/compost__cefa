# Imágenes del proyecto COMPOST CEFA

## Logo de la aplicación

Para que el logo se muestre en la barra lateral (admin y aprendiz), coloca el archivo del logo en esta carpeta con uno de estos nombres:

- **`logo-compost-cefa.webp`** (recomendado)
- **`logo-compost-cefa.png`** (alternativa)

### Pasos

1. Pide a tu compañero el archivo del logo o usa el que tengas del proyecto original.
2. Copia el archivo a: `public/img/logo-compost-cefa.webp` (o `.png`).
3. Si usas solo `.png`, en los layouts se usa `.webp` por defecto; si no tienes el .webp, puedes renombrar tu imagen a `logo-compost-cefa.webp` o convertirla.

Si el archivo no existe, la aplicación mostrará el texto **"COMPOST CEFA"** con un icono como respaldo.

## Otras imágenes

- Las imágenes de **autenticación** (login/registro) van en `public/img/auth/`. Ver `public/img/auth/README.md`.
- Las imágenes de **registros** (pilas, residuos, maquinaria, etc.) se guardan en `storage` al subirlas; no van en esta carpeta.
